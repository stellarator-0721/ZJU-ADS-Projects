# 矩形装箱算法 (Rectangle Packing Algorithms)

## 项目简介

这是一个解决二维矩形装箱问题的算法实现项目。项目包含两种不同的装箱算法：**贪心算法 (Greedy Algorithm)** 和**模拟退火算法 (Simulated Annealing)**。这些算法旨在将一组给定尺寸的矩形尽可能紧凑地装入固定宽度的无限高度条带中，目标是**最小化所需的高度**。

## 主要功能

1. **两种装箱策略**：
   - **贪心算法 (NFDH - Next Fit Decreasing Height)**：快速但可能不是最优解的启发式算法
   - **模拟退火算法**：基于概率的优化算法，有更大机会找到接近最优解

2. **性能评估**：
   - 自动测量算法执行时间
   - 提供详细的性能统计报告

3. **结果输出**：
   - 生成标准化的结果文件
   - 包含算法名称、输入参数、结果高度和运行时间等详细信息

4. **批量处理支持**：
   - 可处理不同规模的输入文件
   - 支持自定义矩形数量和尺寸

## 输入文件格式详解

每个输入文件必须严格按照以下格式：

```
n W
w₁ h₁
w₂ h₂
...
wₙ hₙ
```

**参数说明**：
- `n`：矩形数量（正整数）
- `W`：条带宽度（正整数）
- `wᵢ`：第 i 个矩形的宽度（正整数）
- `hᵢ`：第 i 个矩形的高度（正整数）

**示例** (`data/input_5.txt`)：
```
5 100
30 40
25 35
45 20
15 60
35 25
```
